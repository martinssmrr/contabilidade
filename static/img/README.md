# Diretório de Imagens

## Imagens Necessárias

### 1. logo.png
- **Descrição**: Logo da empresa Gestão 360
- **Local de uso**: Navbar e Hero Section
- **Dimensões recomendadas**: 300x80px (ou proporção similar)
- **Formato**: PNG com fundo transparente
- **Cores**: Deve conter as cores primárias do projeto (azul marinho #1e3a8a, ciano #06b6d4, âmbar #f59e0b)

### 2. popup_home.png
- **Descrição**: Imagem ilustrativa para o popup de contato
- **Local de uso**: Lado esquerdo do modal de contato na home page
- **Dimensões recomendadas**: 600x800px (vertical)
- **Formato**: PNG ou JPG
- **Conteúdo sugerido**: Imagem moderna relacionada a contabilidade, gestão empresarial, ou atendimento ao cliente

## Como adicionar as imagens

1. Salve as imagens neste diretório: `static/img/`
2. Certifique-se de que os nomes dos arquivos estejam exatamente como especificado acima
3. Execute `python manage.py collectstatic` se estiver em produção
4. Recarregue a página (F5) para ver as imagens

## Observações

- Se você não tiver as imagens agora, pode criar placeholders temporários ou usar imagens de exemplo
- As cores do logo foram usadas para definir o esquema de cores do site
- Todas as imagens devem ser otimizadas para web (reduzir tamanho sem perder qualidade)
